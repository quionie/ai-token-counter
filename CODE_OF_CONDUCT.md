# Code of Conduct

This project is committed to providing a respectful, constructive, and harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing opinions and experience levels
- Giving actionable, technical feedback
- Assuming good intent while discussing tradeoffs
- Focusing on what improves the project for users

Examples of unacceptable behavior:

- Harassment, personal attacks, or discrimination
- Trolling, hostility, or deliberately inflammatory behavior
- Publishing private information without permission
- Dismissing contributors in ways that shut down collaboration

## Scope

This code of conduct applies to:

- GitHub issues
- Pull requests
- Discussions related to the project

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer through the repository issue tracker or the repository contact links.
